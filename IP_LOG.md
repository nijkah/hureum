# IP Log

This project follows a clean-room process. Implementation sources must be public platform documentation, open standards, or original design notes.

## Initial Sources

- Electron public documentation for desktop shell concepts.
- MDN WebGPU API documentation for browser GPU access.
- MDN Web Audio API documentation for microphone analysis concepts.
- MDN MediaRecorder documentation for canvas video recording concepts.
- React, Vite, and React Flow public documentation for application structure.

## Prohibited Sources

- TouchDesigner binaries, project files, screenshots, internal APIs, private formats, sample networks, palette contents, icons, and copied operator behavior.
